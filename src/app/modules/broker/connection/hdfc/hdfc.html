Step-by-Step Guide

Steps to Obtain API Key and API Secret for HDFC Securities
  1. Visit the HDFC Securities Developer Portal.
  2. Log in with your ID, password, and OTP.
  3. Accept the Risk Disclosure.
  4. Click Create to make a new app. Enter app name, redirect URL: "{{ brokerConnectRedirectUrl }}" and description, then click Create.
  5. Copy the API Key and API Secret and paste them into the AlphaQuarkTest platform to connect your broker.
